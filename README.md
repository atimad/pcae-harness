# PCAE Harness

PCAE means Persistent Constrained Agentic Engineering Harness.

This project is a cross-platform Python CLI that can be injected into a Git repository to make AI coding safer, resumable, modular, and documentation-driven.

Phase 1 implements only:

- package structure
- CLI skeleton
- `pcae init`
- tests for init
