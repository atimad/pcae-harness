# Skill

## Skill ID

task-transition

## Skill Name

Task Transition

## Skill Type

workflow

## Skill Version

1.0.0

## Skill Status

active

## Human Review Required

true

## Purpose

Support governed task transition planning and verification without hidden state changes.

## Workflow

1. Confirm the active task and next intended task title.
2. Validate continuity against project status and session state.
3. Execute the explicit transition command when requested.
4. Re-run governance checks after the transition.
